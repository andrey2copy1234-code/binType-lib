# binType-lib
библиотека для того чтобы сохранять в файлы бинарный формат данных
## Использование
Вы можете объявить свой тип который будете сохранять бинарно. для этого нужно написать:
```python
class MyBinStruct(BigStructure):
  attr1: type
  attr2: type2
# ...
```
Под type может быть всё что угодно. вот все типы имеющиеся сейчас в библиотеке:
| Название Типа | Что из себя представляет | Диапозон возможностей |
| :--- | :--- | :--- |
| `signedInt8` | целочисленое число хранимое в 8 битах | от -128 до 127 |
| `signedInt16` | целочисленое число хранимое в 16 битах | от -32 768 до 32 767 |
| `signedInt32` | целочисленое число хранимое в 32 битах | от -2 147 483 648 до 2 147 483 647 |
| `signedInt64` | целочисленое число хранимое в 64 битах | от -9 223 372 036 854 775 808 до 9 223 372 036 854 775 807 |
| `signedInt128` | целочисленое число хранимое в 128 битах | от -1.7e38 до 1.7e38 |
| `int8` | целочисленое не отрицательное число хранимое в 8 битах | от 0 до 255 |
| `int16` | целочисленое не отрицательно число хранимое в 16 битах | от 0 до 65 535 |
| `int32` | целочисленое не отрицательное число хранимое в 32 битах | от 0 до 4 294 967 296 |
| `int64` | целочисленое не отрицательное число хранимое в 64 битах | от 0 до 1.8e19 |
| `int128` | целочисленое не отрицательное число хранимое в 128 битах | от 0 до 3.4e38 |
| `Bool` | булевое значение хранимое в 8 битах | только False и True |
| `Char` | символ хранимый в 32 битах (utf-8) | все символы utf-8 включая смалики |
| `String` | utf-8 строку | все символы utf-8 включая смалики и до 4 294 967 296 символов |
| `Bytes` | последовательность байтов | от 0 до 4 294 967 296 байт |
| `CompBytes` | сжатая последовательность байтов | от 0 до 4 294 967 296 сжатых байт |
| `ArrayType([], type)` | не динамический масив из n элементов одного типа | от 0 до бесконечности элементов |
| `DinamicArray([], type)` | динамический масив из n элементов одного типа | от 0 до 4 294 967 296 элементов |
| `Any` | любой python тип | всё |
| `Status` | тип статуса | 0-255 статусов |
| `BigStructure` | пользовательский тип (который мы объявляли) | всё |
Когда вы написали всё типы которые хранит внутри себя MyBinStruct можно создать объект MyBinStruct заполнить его данными вызвать to_bytes и сохранить выданные байты в файл:
```python
with open("MyBinStruct.bin", 'wb') as f:
  f.write(MyBinStructVar.to_bytes())
```
А когда надо будет загрузить MyBinStruct то прочитайте данные после чего вызовите `MyBinStruct.load(bytes)`:
```python
with open("MyBinStruct.bin", 'rb') as f:
  bytes = f.read()
MyBinStructVar = MyBinStruct.load(bytes)
```
Также в библиотеке есть также 2 класса от которых можно наследоваться:
* TextView: при вызове str(type) показывает BigStructure в формате: `typename(attr1=attr1_val, attr2=attr2_val...)`
* AutoCreate: добавляет `__init__` там где вы передаёте в формате `type(attr1_val, attr2_val..., attr3_name=attr3_val)`
## Создания собственых типов
Есть 2 способа создать свой тип:
1. создать из уже имеющихся типов соеденив их при помоши BigStruture
2. создать полностью новый тип
Здесь я разберу второй вареант.

Для начала нам потребуется наследовать свой класс нового типа от типа Structure. от его наследуются все типы. потом надо добавить `__init__` (нужно чтобы библиотека превращала обычный python тип в ваш), `to_bytes` и `from_bytes` а также 2 атрибута: size и represents.
```python
class MyNewType(Structure):
  size = 4 # в байтах. укажите -1 если тип динамического размера
  represents = python_type # укажите python тип который представляет ваш новый тип это поможет отлавливать ошибки в AutoCreate
  def __init__(self, arg):
    # ... тут надо написать преобразователь всех python типов которые относятся к вам в свой тип
  def to_bytes(self):
    return bytes # преобразуйте свой тип в байты
  def from_bytes(self, data: bytes):
    return python_type # здесь надо преобразовать байты в python тип
  # def this_size(self):
  #   return my_size # для динамических типов возврощайте здесь размер
```
